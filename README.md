# Tic Tac Toe

This is a simple Tic Tac Toe game implemented in Java.

## How to Run

1. Compile the Java files:
   ```bash
   javac -d bin src/ticTakToeMain/*.java
   ```
2. Run the main application:
   ```bash
   java -cp bin ticTakToeMain.Gui
   ```

## Technologies Used

* Java
* Swing (for GUI version)
